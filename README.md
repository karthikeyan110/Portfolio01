# Karthikeyan Pandita - Personal Portfolio

Welcome to my personal portfolio repository! This is a modern, responsive single-page web application built to showcase my skills, projects, and professional background as a Computer Science Engineering student.

## 🚀 Features

- **Modern UI/UX**: Designed with a sleek dark theme, custom CSS animations, gradients, and micro-interactions.
- **Projects Showcase**: Highlights my core data science and software engineering projects (e.g., Smart Fraud Detection System, Election Management System).
- **Skills Section**: Categorized technical skills including Languages, Frontend, Backend, Data Science, and Tools.
- **Resume Download**: Integrated direct download for my latest CV.
- **Responsive Layout**: Fully optimized for both desktop and mobile viewing.

## 🛠️ Built With

- **[React](https://reactjs.org/)** - Frontend library for building user interfaces.
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for fast development.
- **Vanilla CSS** - Custom styling, animations, and responsive grids without heavy UI frameworks.

## 💻 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (if you have Git installed):
   ```bash
   git clone https://github.com/karthikeyan110/your-repo-name.git
   ```
   *Alternatively, you can download the code as a ZIP file from GitHub and extract it.*

2. **Navigate to the project directory**:
   ```bash
   cd portfolio-app
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open in browser**:
   Open your browser and navigate to the local URL provided in your terminal (usually `http://localhost:5173/`).
